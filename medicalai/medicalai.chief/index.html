    class medicalai.chief.dataset_prepare.datasetGenFromDataframe(folder, csv_path='.', x_col='name', y_col='labels', targetDim=(224, 224), normalize=False, batch_size=16, augmentation=True, color_mode='rgb', class_mode='sparse', shuffle=True, seed=17)¶

    Bases: object

    Creates Keras Dataset Generator for Handling Large Datasets from DataFrame.

    • Parameters

      • csv_path ‚Äì folder containing train.csv and test.csv.

      • folder ‚Äì The directory must be set to the path where your training images are present.

      • x_col ‚Äì Name of column containing image name, default = name.

      • y_col ‚Äì Name of column for labels, default = labels.

      • targetDim ‚Äì The target_size is the size of your input images to the neural network.

      • class_mode ‚Äì Set binary if classifying only two classes, if not set to categorical, in case of an Autoencoder system, both input and the output would probably be the same image, for this case set to input.

      • color_mode ‚Äì grayscale for black and white or grayscale, rgb for three color channels.

      • batch_size ‚Äì Number of images to be yielded from the generator per batch. If training fails lower this number.
